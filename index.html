Asmos

An ASM kernel project for x86 architecture

Download this project as a .zip file Download this project as a tar.gz file

Welcome to ASMOS.

The project aim is to do an open source operating system in asm to be run on old computers. Lot of people have an old PC and no operating system can be setup on it. So, asmos give them an second life. To get asmos, you have to do :

$ git clone https://github.com/ju34/asmos.git

Compile and launch asmos

For the time, asmos cannot be lunched on computer, but you can try it on bochs. How to do :

$ cd asmos
$ make

Warning : You must have nasm, bochs and gcc installed on your computer.
